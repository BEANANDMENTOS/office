# Agent Office Monitor

Pixel-art static website for monitoring 10 AI agents across a work room and a lounge.

- Mobile-first 9:16 room layout with horizontal scroll snap between rooms.
- Desktop layout shows the two vertical rooms side by side.
- Scene objects and agent characters are separated in `script.js` data arrays.
- No command controls are included; this is a monitoring-only UI.
