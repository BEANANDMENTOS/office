# Agent Office Monitor

Pixel-art static website for monitoring 10 AI agents across a work room and a lounge.

- Mobile-first 9:16 room layout with horizontal scroll snap between composed rooms.
- Desktop layout shows both rooms side by side.
- Rooms, furniture, desks, props, and agents are separate positioned objects in `script.js`.
- No command controls are included; this is a monitoring-only UI.
